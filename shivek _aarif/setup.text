 Set Up the Development Environment
 ( documentation of home page and dashboard)



First, ensure you have Node.js and npm installed. These are essential for managing dependencies and running your React or Vue.js application.
-------------------------------------------------------------------------------------------------------------------------------------
Install Node.js and npm
-------------------------------------------------------------------------------------------------------------------------------------
React Setup:

bash
npx create-react-app my-app
cd my-app
npm start
Vue.js Setup:

-------------------------------------------------------------------------------------------------------------------------------------

bash
Copy code
npm install -g @vue/cli
vue create my-app
cd my-app
npm run serve
